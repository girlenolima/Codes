Singleton é um (anti-)padrão de projeto de software (do inglês Design Pattern). Este padrão garante a existência de apenas uma instância de uma classe, mantendo um ponto global de acesso ao seu objeto.


Problemas
Como posso garantir que uma classse tenha apenas uma instancia?

Como fazer como que esta instancia unica possa ser acessivel globalmente ?

Soluçao
Esconder o construtor dessa classe

Definir um ponto de criaçao estatico

Retornar uma instancia unica
